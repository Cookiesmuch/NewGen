Hardware Scheduling Pipeline

Thread Director operates in a continuous decision loop, monitoring system state and making real-time arbitration decisions. Each step executes in hardware, ensuring deterministic latency.

01
Thread Classification

Monitor instruction mix, branch patterns, and memory access profiles for all runnable threads. Classify workload as latency-bound, throughput-bound, or memory-bound.

02
LPNPU Prediction

LPNPU co-pilot analyzes thread runqueue state using lightweight on-chip AI. Predicts workload class and optimal core target before OS scheduler becomes aware of runnable threads.

03
Thermal & Power Analysis

Query per-core thermal headroom and system power budget. Determine which core classes have available capacity for new threads without violating TDP limits.

04
Core Selection & Dispatch

Select optimal core class based on workload profile, thermal budget, and power state. Signal OS scheduler with dispatch hint. Pre-stage core frequency and voltage for incoming thread.

05
Continuous Monitoring

Monitor thread execution in real-time. If workload characteristics change, migrate thread to more appropriate core class. Coordinate V8 HyperBOOST sequencing for UHP burst windows.

LPNPU Co-Pilot Mode

Predictive AI-Assisted Dispatch

New in Eventide: the Hardware Thread Director operates in co-pilot mode with the always-on Low Power NPU. The LPNPU (88 TOPS, Intel 06E) runs a lightweight on-chip AI scheduling model that predicts workload class before threads become runnable.

By analyzing instruction patterns, memory access behavior, and historical execution profiles, the LPNPU feeds predictive dispatch signals to the Thread Director tile. This allows proactive core preparation—frequency scaling, voltage adjustments, and cache pre-warming—before the OS scheduler even sees the thread.

The result: smarter core selection, fewer thermal missteps, and better sustained performance over time. The LPNPU learns user behavior patterns, adapting its predictions to match real-world usage without manual tuning.
