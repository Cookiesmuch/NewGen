Three Foundational Roles

01

Always-On Inference

Continuous, zero-CPU wake-on-voice keyword detection using Gaussian Mixture Model (GMM) phoneme analysis. The LPNPU monitors ambient audio in real-time, detects user intent through keyword confidence thresholds, and routes SONAR presence signals—all without involving the main processor or waking the LPE cluster.

02

Hardware Thread Director Co-Pilot

Predictive task dispatch co-processor running lightweight AI models to forecast task runqueue states before threads become schedulable. The LPNPU signals the UHP Solar Eclipse cores to pre-stage frequency targets, delivering sub-100ns arbitration advantage over OS scheduler latency.

03

System Intelligence Layer

Persistent behavioral AI that learns user habits and system patterns. Predicts app pre-staging needs, recommends dynamic power profiles, and orchestrates tile wake sequences. Runs continuously, shaping the system's response before user demand is visible to the OS.

WiFi SONAR: Presence Sensing

Doppler-Shift Presence Detection

SONAR (Signal Observation and Nearfield Analysis Radar) leverages the LPNPU's always-on compute to analyze WiFi signal reflections in real-time. By measuring Doppler shifts in reflected RF energy, the LPNPU detects human presence within a ~3m radius—without requiring new hardware, external cameras, or privacy intrusions.

When a user approaches the device, SONAR wakes the display from black screen at sub-100ms latency. As the user moves away, the display dims intelligently. The system learns spatial occupancy patterns, enabling predictive wake sequencing and context-aware power state transitions.

This tight coupling between WiFi analog front-end and LPNPU compute creates a new class of ambient awareness—proximity sensing without the power cost of dedicated proximity hardware.
